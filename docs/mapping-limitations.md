# Mapping Limitations

本项目的地图功能是简单的 occupancy grid demo，不是完整 SLAM。

## 当前估计方式

- 根据当前电机命令估计小车运动
- 根据时间积分更新位姿
- 使用超声波和红外传感器更新局部占据栅格

## 主要限制

- 没有轮速编码器
- 没有 IMU
- 没有闭环定位
- 没有激光雷达
- 没有视觉 SLAM
- 电池电压变化会影响实际速度
- 地面摩擦会影响轨迹
- 左右电机差异会导致偏航误差
- 长时间运行后位姿漂移是预期现象

## 适用场景

- 教学演示
- 传感器融合入门
- 手动驾驶可视化
- 简单避障调试

## 不适用场景

- 精确导航
- 长时间自主建图
- 可靠定位
- 安全关键应用
